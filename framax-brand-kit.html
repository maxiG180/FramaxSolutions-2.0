06 — Brand Rules
Do's & Don'ts
✓ DO
Use Bebas Neue for all headlines
Use #2563eb as the single brand color
Keep backgrounds dark (#0d0d0d)
Use DM Mono for tags, metadata, labels
Maintain generous whitespace
Use the 8px spacing scale consistently
✕ DON'T
Use more than 3 colors in one layout
Use Inter, Roboto, or Arial
Use gradients — keep colors flat
Use light backgrounds for internal tools
Mix spacing values outside the scale
Use more than one accent color per page
